## ğŸ“– About the Project

**Vibe Styles** is a complete visual reference for designers and developers working with UI/UX. The project brings together 256 design styles â€” from Glassmorphism to Cyberpunk, from Brutalism to Kawaii â€” each implemented as a standalone landing page with HTML, Tailwind CSS, and vanilla JavaScript.

The differentiator: each style comes with a **copyable prompt** optimized for generative AI tools (ChatGPT, Claude, Gemini, etc.), allowing you to instantly reproduce any style in your projects.

### Inspiration

It all started with a [Reddit post](https://www.reddit.com/r/notebooklm/comments/1r2p1bb/40_notebooklm_infographic_styles_specification/) by [Paolo Cortez](https://www.reddit.com/user/Paolo-Cortez/), who shared a collection of infographic style prompts for NotebookLM. The specifications were so well-structured â€” with color palettes, typography, textures, and tags â€” that they could easily serve as style specifications for landing pages.

This reminded me of [ui-ux-pro-max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill), a skill that also maintains a [curated list of UI/UX styles](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill/blob/main/src/ui-ux-pro-max/data/styles.csv) with a focus on practical implementation.

I also used [Manus AI](https://manus.im/app) to evolve the lists, jumping from 89 to around 200 styles.

I merged the sources and used [Kiro](https://kiro.dev) with multi-agents to generate dynamic prompts and create the nearly 200 demonstration pages in an automated way.

I also tested it in [Cline](https://cline.bot/) and [Claude Code](https://claude.com/product/claude-code) and it worked great too.

The intention is to be simple and straightforward: to help you choose the right style for your SaaS or weekend project.

### Why does Vibe Styles exist?

- **Quick reference** â€” Browse 256 styles visually without leaving the browser
- **Ready-made prompts** â€” Copy and paste into any generative AI to replicate the style
- **Real code** â€” Each style is a functional implementation, not just a mockup
- **Zero dependencies** â€” Static HTML + Tailwind CDN + Google Fonts. No build, no framework
- **SEO-friendly** â€” URLs with semantic slugs, JSON-LD Schema Markup, Open Graph, sitemap

## ğŸš€ How to Use

### Online Navigation

1. Access **[vibe.ft.ia.br](https://vibe.ft.ia.br/)**
2. Select a style from the navigation bar dropdown
3. View the live demonstration in the central panel
4. Click **"Copy Prompt"** to copy the style prompt
5. Paste the prompt into ChatGPT, Claude, Gemini, or any generative AI

### Local Execution

```bash
# Clone the repository
git clone https://github.com/fabricioctelles/vibe-styles.git
cd vibe-styles

# Serve the static files (any HTTP server works)
python3 -m http.server 8000 --directory app
# or
npx serve app
# or
php -S localhost:8000 -t app
```

Access `http://localhost:8000` in your browser.
