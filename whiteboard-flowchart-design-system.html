Tipografia

4 fontes do Google Fonts formam a hierarquia completa. A ilusão de "escrito à mão" precisa equilibrar personalidade com legibilidade, especialmente em mobile.

Títulos Principais — "Canetão grosso"

Permanent Marker — ABCDEFG 123

Uso: H1, títulos de path, títulos de seção. Sempre uppercase. Size: 2rem–3.5rem.

Corpo / Listas — "Caneta média"

Kalam — The quick brown fox jumps over the lazy dog. 0123456789

Uso: body text, bullets, descrições em cards. Mixed case. Size: 0.85rem–1.1rem. Line-height: 1.55.

Labels & UI — "Escrita limpa"

Patrick Hand — Step Labels & Badges

Uso: step labels, badges, subtítulos menores. Uppercase. Size: 0.75rem–0.9rem.

Notas / Anotações — "Escrita corrida"

Caveat — Anotações rápidas, notas de rodapé, itálico editorial

Uso: captions, notas, annotations. Italic. Size: 0.9rem–1.2rem.

Seção

Ícones Doodle & Conectores

Ícones: SVG inline, stroke-only (sem fill, exceto highlight opcional). Stroke-width: 2px, linecap: round, linejoin: round. Tamanho: 24–40px. Estilo "rabisco rápido" — linhas não perfeitamente retas.

Conectores: SVG paths com curvas Bézier. Pontas de seta triangulares via <marker>. Animação draw-in (stroke-dasharray + dashoffset) para revealing progressivo na landing page.

Tip: Use Rough.js para gerar paths com "jitter" automático, criando curvas que parecem desenhadas à mão.

Seção

Técnicas CSS Chave

Sombras Hard

box-shadow: 3px 3px 0px rgba(26,26,26,0.15);

Zero blur. Offset sólido. Simula "papel colado" sobre o quadro. Variante forte: 5px 5px para caixas de resultado.

Rotações Orgânicas

transform: rotate(-0.8deg);

Cada elemento recebe rotação aleatória entre -1° e 1°. Nunca repita o mesmo ângulo em cards adjacentes.

Textura de Whiteboard

Sobreposição de radial-gradients com opacidade baixa (0.1–0.3) para simular reflexo e leve variação tonal da superfície. Opcional: imagem de textura com mix-blend-mode.

Bordas Irregulares

Opção A: SVG filter feTurbulence + feDisplacementMap com scale baixo (1–2px). Opção B: Rough.js para bordas com jitter automático. Opção C: clip-path polygon com pontos levemente deslocados.

Checklist de Implementação

Obrigatório

  • Google Fonts: Permanent Marker + Kalam + Patrick Hand + Caveat
  • CSS custom properties para todos os tokens
  • Sombras hard (blur: 0) em todos os cards
  • Rotações aleatórias (-1° a 1°) em cada card
  • Moldura metálica (gradient cinza) envolvendo o whiteboard
  • Fundo branco-gelo com textura sutil
  • Bordas pretas sólidas 2–3px
  • Responsivo: grid → stack em ≤ 900px

Opcional (Upgrade)

  • Rough.js para bordas com jitter procedural
  • Animação draw-in nos conectores SVG
  • Hover: elevação + sombra intensificada
  • Textura de whiteboard via background-image
  • Filtro SVG feTurbulence para irregularidade
  • Animação @keyframes wiggle sutil no load
  • Sticky notes com pseudo-element de fita adesiva
  • Intersection Observer para reveal-on-scroll
